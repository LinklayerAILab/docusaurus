# Quick Start

Get up and running with LinkLayerAI DApp in minutes.

## 5-Minute Setup

### Step 1: Connect Your Wallet

1. Click "Connect Wallet" button
2. Choose your wallet provider
3. Approve the connection

### Step 2: Select Exchange

1. Navigate to the Exchange section
2. Choose your preferred exchange (Binance, OKX, Bybit, etc.)
3. Connect your API keys

### Step 3: Configure Your Agent

1. Select an agent type (Alpha, Perps, or Trading)
2. Set your risk parameters
3. Configure trading strategies

### Step 4: Start Trading

1. Make a deposit
2. Set up your trading parameters
3. Monitor performance through the dashboard

## Your First Trade

### Manual Trade

1. Go to the Trading section
2. Select your trading pair
3. Enter amount and parameters
4. Execute the trade

### Agent-Assisted Trade

1. Choose your agent
2. Set your strategy preferences
3. Let the agent analyze and execute
4. Monitor results

## Important Tips

- **Start Small**: Begin with small amounts to test the system
- **Use Testnet**: Practice on testnet before using real funds
- **Set Limits**: Always set stop-loss and take-profit levels
- **Monitor**: Keep an eye on your active positions

## Common Issues

**Q: Why is my transaction pending?** A: Network congestion may cause delays. Check the gas fees and try again if needed.

**Q: Can I use multiple exchanges?** A: Yes! You can connect to multiple exchanges simultaneously.

**Q: Is my funds safe?** A: Yes, we use non-custodial approach. You always control your funds.
