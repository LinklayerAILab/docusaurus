# Extension Basic Usage

Learn the fundamentals of using the LinkLayerAI Chrome Extension.

## Getting Started

### Opening the Extension

There are several ways to access the extension:

1. **Click the toolbar icon** - LinkLayerAI logo in browser toolbar
2. **Keyboard shortcut** - `Alt+L` (customizable in settings)
3. **Context menu** - Right-click on page → "LinkLayerAI"

### Main Interface Overview

The extension interface consists of:

- **Header**: Account info, settings, notifications
- **Dashboard**: Overview of portfolio and positions
- **Trading Panel**: Execute trades and manage orders
- **Agent Panel**: AI agent controls and signals

## Daily Workflow

### 1. Morning Check

**Market Overview**

- Open the extension
- Check portfolio value and P&L
- Review overnight market movements
- Read agent recommendations

### 2. Monitoring Throughout the Day

**Price Alerts**

- Set up price alerts for key levels
- Monitor alert notifications
- Review market impact
- Decide on action

**Agent Signals**

- Watch for AI agent recommendations
- Evaluate signal quality
- Check confidence levels
- Execute if aligned with strategy

### 3. Executing Trades

**Manual Trading**

1. Click "New Trade" button
2. Select exchange and trading pair
3. Choose order type (Market/Limit)
4. Enter amount and price
5. Set stop-loss and take-profit
6. Review and confirm

**Agent-Assisted Trading**

1. Go to Agent Panel
2. Select desired agent
3. Review agent's recommendation
4. Adjust parameters if needed
5. Approve execution
6. Monitor position

## Key Features

### Quick Trade

**For fast execution:**

1. Click "Quick Trade" button
2. Pre-configured settings load
3. Enter amount
4. One-click execution

### Portfolio View

**What you'll see:**

- Total portfolio value
- Asset allocation breakdown
- P&L by position
- Win/Loss ratio

**Actions:**

- Click position for details
- Adjust position size
- Close position
- View trade history

## Common Tasks

### Checking Balance

1. Open extension
2. View "Balance" section
3. Filter by exchange or asset

### Placing an Order

1. Click "Trade" tab
2. Select exchange
3. Choose trading pair
4. Select order type
5. Enter parameters
6. Confirm order

### Managing Positions

1. Go to "Positions" tab
2. Click desired position
3. Options:
   - **Add to position**: Increase size
   - **Reduce position**: Partial close
   - **Close position**: Full exit

## Tips for Efficient Usage

### Keyboard Shortcuts

- `Alt+L`: Open extension
- `Alt+T`: Quick trade
- `Alt+P`: Positions
- `Alt+A`: Agent panel

### Best Practices

1. **Start Simple**

   - Use basic features first
   - Gradually adopt advanced features
   - Practice with small amounts

2. **Stay Organized**

   - Use consistent naming conventions
   - Keep trade notes
   - Maintain templates

3. **Monitor Regularly**

   - Check positions frequently
   - Review agent performance
   - Adjust settings as needed

4. **Security First**
   - Lock extension when not in use
   - Use 2FA
   - Review connected devices
   - Keep API keys secure

## Troubleshooting Basic Issues

### Extension Not Loading

- Refresh the page
- Restart browser
- Check internet connection

### Data Not Updating

- Check API connection status
- Verify API keys are valid
- Try manual refresh

### Order Not Executing

- Verify sufficient balance
- Check order parameters
- Review exchange status
