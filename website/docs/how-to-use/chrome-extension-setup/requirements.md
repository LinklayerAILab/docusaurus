# Extension Requirements

Before installing the LinkLayerAI Chrome Extension, ensure your system meets these requirements.

## System Requirements

### Browser

- **Google Chrome**: Version 90 or higher
- **Brave Browser**: Version 1.20 or higher
- **Microsoft Edge**: Version 90 or higher (Chromium-based)

### Operating System

- **Windows**: Windows 10 or later
- **macOS**: macOS 10.13 (High Sierra) or later
- **Linux**: Ubuntu 18.04 or equivalent

### Hardware

- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 100MB free space
- **Network**: Stable internet connection

## Account Requirements

### Exchange Accounts

You'll need active accounts on at least one of these exchanges:

- [Binance](https://www.binance.com)
- [OKX](https://www.okx.com)
- [Bybit](https://www.bybit.com)
- [Bitget](https://www.bitget.com)

### API Keys

- Read permissions (required)
- Trading permissions (optional, for trading features)
- IP whitelist (recommended for security)

## Security Recommendations

### API Key Security

1. **Never share** your API keys
2. **Use IP whitelist** to restrict access
3. **Rotate keys regularly** (every 3-6 months)
4. **Use different keys** for different applications

### Browser Security

- Keep your browser updated
- Use a reputable antivirus
- Enable pop-up blockers
- Use a password manager

## Checking Your Setup

### Verify Browser Version

1. Open Chrome
2. Go to `chrome://version`
3. Check if version meets requirements

### Check Exchange Access

1. Log into your exchange account
2. Navigate to API management
3. Verify you can create API keys

## Ready to Install?

Once you've verified all requirements, proceed to [Setup Guide](./setup).
