# DApp Installation

Learn how to install and set up the LinkLayerAI DApp.

## System Requirements

- **Browser**: Chrome, Firefox, Safari, or Edge (latest version)
- **Wallet**: MetaMask, WalletConnect, or other Web3 compatible wallet
- **Network**: Stable internet connection

## Installation Steps

### 1. Install a Web3 Wallet

If you don't have a Web3 wallet, we recommend:

- **MetaMask**: [Download here](https://metamask.io/)
- **WalletConnect**: [Learn more](https://walletconnect.com/)

### 2. Connect Your Wallet

1. Visit the LinkLayerAI DApp
2. Click "Connect Wallet" in the top right corner
3. Select your preferred wallet
4. Approve the connection request

### 3. Configure Settings

- Select your preferred network (Ethereum, BSC, Polygon, etc.)
- Set up your trading preferences
- Configure security settings

## Troubleshooting

### Wallet Connection Issues

If you're having trouble connecting your wallet:

1. Make sure your wallet is unlocked
2. Check that you're on the correct network
3. Clear your browser cache and try again
4. Ensure your browser allows Web3 connections

## Next Steps

Once installed, explore the [Dashboard](./dashboard) to start using the DApp.
