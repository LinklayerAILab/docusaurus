# OKX App API Configuration

## Configuration Steps

1. On the homepage, click the top left corner.

<img src="https://cdn.linklayer.ai/webpage/doc/api/okxapp/okxapp1_compressed.png"  width="350" />

2. Scroll to the bottom of the page and click API in the More section.

<img src="https://cdn.linklayer.ai/webpage/doc/api/okxapp/okxapp2_compressed.png"  width="350" />

3. Click "Create API key".

<img src="https://cdn.linklayer.ai/webpage/doc/api/okxapp/okxapp3_compressed.png"  width="350" />

4. Set the API name and password, and select read-only permissions.

<img src="https://cdn.linklayer.ai/webpage/doc/api/okxapp/okxapp4_compressed.png"  width="350" />

5. Complete the security verification.

<img src="https://cdn.linklayer.ai/webpage/doc/api/okxapp/okxapp5_compressed.png"  width="350" />

6. API Key has been successfully created.

<img src="https://cdn.linklayer.ai/webpage/doc/api/okxapp/okxapp6_compressed.png"  width="350" />
