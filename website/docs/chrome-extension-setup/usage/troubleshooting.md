# Troubleshooting

Common issues and solutions for the LinkLayerAI Chrome Extension.

## Installation Issues

### Extension Won't Install

**Problem**: Installation fails or extension doesn't appear

**Solutions**:

1. Check Chrome version (must be 90+)
2. Clear browser cache
3. Disable conflicting extensions
4. Try downloading again
5. Restart browser

### Extension Not Showing

**Problem**: Extension icon doesn't appear in toolbar

**Solutions**:

1. Click puzzle icon in toolbar
2. Find LinkLayerAI and pin it
3. Check if extension is enabled
4. Restart Chrome

## Connection Issues

### Can't Connect Exchange

**Problem**: Exchange connection fails

**Solutions**:

1. Verify API keys are correct
2. Check API key permissions
3. Ensure IP whitelist includes your IP
4. Test exchange API status
5. Try reconnecting

### Connection Drops Frequently

**Problem**: Extension keeps disconnecting

**Solutions**:

1. Check internet connection
2. Disable VPN or proxy temporarily
3. Clear extension cache
4. Re-enter API keys
5. Contact exchange support if issue persists

## Trading Issues

### Order Not Executing

**Problem**: Order fails or rejected

**Solutions**:

1. Check sufficient balance
2. Verify order parameters
3. Check exchange status
4. Review API trading permissions
5. Check for error messages

### Unexpected Slippage

**Problem**: Orders execute at different prices

**Solutions**:

1. Use limit orders instead of market
2. Reduce order size
3. Check market liquidity
4. Avoid volatile market conditions

## Data Issues

### Balance Not Updating

**Problem**: Balance shows old or incorrect data

**Solutions**:

1. Click refresh button
2. Check connection status
3. Verify API keys are valid
4. Check exchange rate limits
5. Clear extension cache

### Charts Not Loading

**Problem**: Price charts don't display

**Solutions**:

1. Check internet connection
2. Disable ad blockers
3. Try different browser
4. Clear browser cache
5. Update extension

## Agent Issues

### Agent Not Responding

**Problem**: AI agent doesn't provide signals

**Solutions**:

1. Check agent is enabled
2. Verify connection to exchange
3. Check account has sufficient balance
4. Review agent configuration
5. Restart extension

### Poor Agent Performance

**Problem**: Agent recommendations losing money

**Solutions**:

1. Review agent settings
2. Adjust risk parameters
3. Check market conditions
4. Consider different agent type
5. Use manual trading temporarily

## Notification Issues

### Not Receiving Alerts

**Problem**: Price alerts or notifications not working

**Solutions**:

1. Check notification permissions
2. Verify alert conditions are met
3. Test alert creation
4. Check Telegram bot connection
5. Review notification settings

### Too Many Notifications

**Problem**: Receiving excessive alerts

**Solutions**:

1. Adjust alert thresholds
2. Disable less important alerts
3. Set quiet hours
4. Use digest mode instead
5. Filter by importance

## Performance Issues

### Extension Running Slow

**Problem**: Extension is laggy or slow

**Solutions**:

1. Reduce data refresh rate
2. Clear cache and history
3. Disable unused features
4. Close other browser tabs
5. Update browser and extension

### High Memory Usage

**Problem**: Browser using too much memory

**Solutions**:

1. Reduce history retention
2. Disable real-time updates
3. Clear cached data
4. Use compact mode
5. Restart browser regularly

## Security Issues

### Suspicious Activity

**Problem**: Unusual trades or account changes

**Solutions**:

1. Immediately disconnect exchanges
2. Change API keys
3. Review account activity
4. Enable 2FA
5. Contact support

### API Key Compromised

**Problem**: API key may be stolen

**Solutions**:

1. Delete API key immediately
2. Generate new API key
3. Update extension with new key
4. Enable IP whitelist
5. Review recent trades

## Quick Fixes

### Reset Extension

If nothing else works:

1. Go to `chrome://extensions`
2. Find LinkLayerAI extension
3. Click "Remove"
4. Reinstall extension
5. Reconnect exchanges

### Clear All Data

**Warning**: This will reset all settings

1. Go to extension settings
2. Click "Clear Data"
3. Confirm action
4. Reconfigure extension

### Fresh Start

Complete reset:

1. Export important data
2. Clear extension data
3. Uninstall extension
4. Reinstall from scratch
5. Import configuration

## Common Error Messages

### "Invalid API Key"

- Check for typos
- Regenerate key
- Verify permissions

### "Insufficient Balance"

- Add funds to exchange
- Check correct currency
- Verify account status

### "Connection Timeout"

- Check internet
- Disable VPN
- Try different network

### "Rate Limit Exceeded"

- Reduce refresh rate
- Slow down requests
- Upgrade exchange tier

2. Verify all requirements are met
3. Try on a different device or browser
4. Reach out to support with detailed information
