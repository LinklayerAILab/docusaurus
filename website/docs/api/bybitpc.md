# Bybit PC 端 API 配置

## 配置步骤

1.登录您的Bybit账户，点击右上角的用户图标，在下拉菜单中选择“API”。


![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc1_compressed.png)

2.单击“Create New Key”。


 ![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc2_compressed.png)

3.点击“System-generated API Keys”。


 ![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc3_compressed.png)

4.输入名称，选中只读框，选择所有项目


![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc4_compressed.png)

5.完成安全验证。


![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc5_compressed.png)

6.API Key已成功创建。


![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc6_compressed.png)
