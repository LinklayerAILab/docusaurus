# Binance PC API Configuration

## Configuration Steps

1. Log in to your Binance account, click the user icon in the top right corner, and select "Dashboard".

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc1_compressed.png)

2. Click "API Management" in the left navigation bar.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc2_compressed.png)

3. Click "Create API".

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc3_compressed.png)

4. Select "System generated" and click next.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc4_compressed.png)

5. Set the label for your API Key.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc5_compressed.png)

6. Complete the security verification.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc6_compressed.png)

7. After verification, the API restriction defaults to read-only.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc7_compressed.png)
