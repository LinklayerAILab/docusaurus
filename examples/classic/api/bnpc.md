1.登录您的币安账户，点击右上角的用户图标，选择”Dashboard”。

Log in to your Binance account, click the user icon in the upper right corner, and select "Dashboard".

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc1.png)

2.点击左侧导航栏的”API Management”。

Click "API Management" in the left navigation bar.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc2.png)

3.点击”Create API”。

Click”Create API key”.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc3.png)

4.选择 “System generated”,点击下一步。

Select “System generated” and click Next.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc4.png)

5.设置API Key的标签。

Set the label of the API Key.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc5.png)

6.完成安全验证。

Complete security verification.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc6.png)

7.验证通过后，API限制默认选择只读。

After successful verification, the API restriction defaults to read-only.

![](https://cdn.linklayer.ai/webpage/doc/api/bnpc/bnpc7.png)
