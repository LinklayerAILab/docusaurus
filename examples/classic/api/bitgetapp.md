1.点击Bitget App首页左上角

Click the top left corner of the Bitget App homepage.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp1.png"  width="350" />

2.点击 More Services

Click More Services

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp2.png"  width="350" />

3.点击API keys

Click API keys

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp3.png"  width="350" />

4.点击“Create a new API key”

Click “Create a new API key”

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp4.png"  width="350" />

5.选择 “Automatically generated APl keys”

Select “Automatically generated APl keys”

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp5.png"  width="350" />

6.设置API key名称及Passphrase，权限选择只读，并勾选只读权限下的所有项目

Set the API key name and Passphrase, select read-only permissions, and select all items under read-only permissions.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp6.png"  width="350" />

7.完成安全验证。

Complete security verification.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp7.png"  width="350" />

8.API Key已成功创建。

API Key has been successfully created.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bitgetapp/bgapp8.png"  width="350" />
