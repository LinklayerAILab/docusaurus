1.登录您的Bitget账户，点击右上角的用户图标，点击”API Keys”。

Log in to your Bitget account, click the user icon in the upper right corner, and then click "API Keys".

![](https://cdn.linklayer.ai/webpage/doc/api/bitgetpc/bgpc1.png)

2\.点击”Create API key”。

Click "Create API key".

![](https://cdn.linklayer.ai/webpage/doc/api/bitgetpc/bgpc2.png)

3\.选择”System-generated API key”。

Select “System-generated API key”.

![](https://cdn.linklayer.ai/webpage/doc/api/bitgetpc/bgpc3.png)

4\.设置API key名称及密码 ,权限选择只读,勾选“Select all”

Set the API key name and password, select read-only permissions, and check "Select all".

![](https://cdn.linklayer.ai/webpage/doc/api/bitgetpc/bgpc4.png)

5.完成安全验证。

Complete security verification.

![](https://cdn.linklayer.ai/webpage/doc/api/bitgetpc/bgpc5.png)

6.API Key已成功创建。

API Key has been successfully created.

![](https://cdn.linklayer.ai/webpage/doc/api/bitgetpc/bgpc6.png)
