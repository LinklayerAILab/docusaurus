1.登录您的OKX账号，点击右上角的人形图标，下拉菜单里选择"API"。

Log in to your OKX account, click the person icon in the upper right corner, and select "API" from the drop-down menu.

![](https://cdn.linklayer.ai/webpage/doc/api/okkxpc/okxpc1.png)

2.点击“Create API”。

Click”Create API key”.

![](https://cdn.linklayer.ai/webpage/doc/api/okkxpc/okxpc2.png)

3.设置API Key名称及密码,权限选择只读。

Set the API Key name and password, and select read-only permissions.

![](https://cdn.linklayer.ai/webpage/doc/api/okkxpc/okxpc3.png)

4、API Key已成功创建。

API Key has been successfully created.

![](https://cdn.linklayer.ai/webpage/doc/api/okkxpc/okxpc4.png)

![](https://cdn.linklayer.ai/webpage/doc/api/okkxpc/okxpc5.png)
