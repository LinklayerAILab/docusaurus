1.登录您的Bybit账户，点击右上角的用户图标，在下拉菜单中选择“API”。

Log in to your Bybit account, click the user icon in the upper right corner, and select "API" from the drop-down menu.

![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc1.png)

2.单击“Create New Key”。

Click”Create New Key”.

 ![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc2.png)

3.点击“System-generated API Keys”。

Click”System-generated API Keys”.

 ![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc3.png)

4.输入名称，选中只读框，选择所有项目

Enter a name, check the read only box, and select all items

![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc4.png)

5.完成安全验证。

Complete security verification.

![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc5.png)

6.API Key已成功创建。

API Key has been successfully created.

![](https://cdn.linklayer.ai/webpage/doc/api/bybitpc/bbpc6.png)
