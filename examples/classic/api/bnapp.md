1.在币安app首页，点击左上角。

On the Binance app homepage, click the top left corner.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp1.png"  width="350" />

2.点击 “More Services”。

Click “More Services”.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp2.png"  width="350" />

3.顶部搜索框里输入API进行搜索。

Enter "API" in the search box at the top to search.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp3.png"  width="350" />

4.点击”API Management”。

Click "API Management".

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp4.png"  width="350" />

5.点击”Create API”。

Click "Create API".

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp5.png"  width="350" />

6.选择”System generated”

Select "System generated"

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp6.png"  width="350" />

7.设置API key的标签

Set the label for the API key

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp7.png"  width="350" />

8.完成安全验证。

Complete security verification.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp8.png"  width="350" />

9.验证通过后，API限制默认选择只读。

After successful verification, the API restriction defaults to read-only.

<img src="https://cdn.linklayer.ai/webpage/doc/api/bnapp/bnapp9.png"  width="350" />
